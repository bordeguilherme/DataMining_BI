  <h1 align="center">Mineração de Dados Estruturados</h1>
  </br>

  <h2>Descrição</h2>
  Este repositório foi criado para incluir o trabalho correspondente à disciplina de Mineração de Dados Estruturados do curso de Ciência de Dados e Business Intelligence. 
  O objetivo principal é aplicar técnicas de clusterização e análise de correlação para explorar os padrões presentes em diferentes tipos de vinho.

  <h2>Avaliação</h2>
  O projeto consiste em aplicar o algoritmo K-means para segmentar os dados do dataset e interpretar a relação entre variáveis químicas e visuais dos vinhos. Também envolve a análise de correlação para identificar pares de variáveis úteis para discriminar os clusters.

  <h2>Requisitos</h2>
  <h3>Importação e Visualização de Dados</h3>
  <li>Importar e visualizar os dados do dataset Wine disponível na biblioteca sklearn.</li>
  <li>Familiarizar-se com as variáveis do dataset e descrever seu significado.</li>

  <h3>Clusterização e Análise</h3>
  <li>Aplicar o algoritmo K-means para criar três clusters, adicionando uma nova coluna no DataFrame com as informações do cluster.</li>
  <li>Calcular a matriz de correlação e identificar variáveis com correlação moderada para análise adicional.</li>

  <h3>Visualização</h3>
  <li>Criar gráficos de dispersão para representar os clusters utilizando variáveis correlacionadas.</li>
  <li>Interpretar os padrões de clusters observados.</li>

  <h2>Conclusão</h2>
  A análise de clusters permitiu observar padrões significativos entre os diferentes tipos de vinho com base em suas características químicas e visuais. As variáveis selecionadas através da análise de correlação ajudaram a identificar distinções claras entre os clusters.
